## ⚙️ System requirements

TranslatePDF works on modern versions of Windows. We suggest Windows 10 or Windows 11. Ensure your computer has at least 4 gigabytes of memory. You need enough disk space to store your translated PDF files. Connect to the internet during the initial setup to download translation data.

## 📝 Translating your first document

Open the TranslatePDF application. You see a clear, simple window. Drag your PDF file into the drop zone in the center of the screen. Alternatively, click the browse button to select a file from your hard drive. 

After you choose a file, select your target language from the list. The tool recognizes the input language automatically. You do not need to set the source language yourself. 

Review the settings panel on the right side of the screen. You can choose to maintain full formatting here. Leave the default settings checked if you want the best results.

Click the translate button at the bottom of the window. A bar shows your progress. The process takes different amounts of time based on the file size. Do not close the window while the bar moves.

The tool saves the document to the same folder as your original file once the translation finishes. You see a notification on the screen informing you of success.

## 🛠️ Handling common issues

Sometimes a PDF contains scanned images instead of text. Scanned files require Optical Character Recognition. TranslatePDF detects this need automatically. The process might take longer for long books that consist of many images.

Ensure your PDF is not password protected. The software needs access to the text inside the file to change the language. You must remove the password before you open such a file in this application.

Check your internet connection if the translation request fails. The tool sends requests to a secure server to get results. A stable connection keeps this process moving fast.

## 📉 Frequently asked questions

Do I need other software to run TranslatePDF? 
No. Everything stays contained within the installer you downloaded. You do not need Java or external libraries.

Does the software change my original file? 
No. TranslatePDF creates a new document. Your original file remains unchanged in your folder.

Can I translate many files at once?
Yes. You can add a batch of files to the queue. The application processes these one by one.

Why do the fonts look slightly different? 
We match your fonts as closely as possible. Some fonts lack support for specific foreign characters. In those cases, the software picks a similar font so your layout stays neat.

Does this tool work offline? 
The core engine requires an internet connection to perform translations. You must remain online to use the features.

## 📂 Keeping your data private

Your documents get sent to our translation engine to produce results. We transfer these files over encrypted connections. We do not store your files on our servers after the process ends. Your data stays private throughout the translation workflow.

## 🏷️ Technical topics

This application uses the PyMuPDF library to read and edit your documents. It provides support for complex layouts. We built the tool to serve literary translation needs. It respects the typography of your books. You can use this for localization of manuals or digital e-books. The focus remains on quality and style retention.
